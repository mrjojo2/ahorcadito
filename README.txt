🎮 AHORCADITO MULTIJUGADOR - GUÍA COMPLETA 🎮

REQUISITOS:
- Node.js instalado (versión 14 o superior)
- Todos los dispositivos en la MISMA RED LOCAL

INSTALACIÓN:
1. Extraer todos los archivos en una carpeta
2. Abrir terminal/CMD en esa carpeta
3. Ejecutar: npm install
4. Ejecutar: npm start

CÓMO JUGAR:
1. El servidor mostrará una IP (ej: http://192.168.1.100:3000)
2. Todos los jugadores acceden a esa IP desde su navegador
3. Cada jugador ingresa su nombre
4. Máximo 10 jugadores por partida

REGLAS DEL JUEGO:
✓ Por turnos, cada jugador intenta adivinar la palabra letra por letra
✓ Si aciertas una letra, SIGUES jugando
✓ Si fallas una letra, el turno pasa al siguiente jugador
✓ Si completas la palabra, ganas tantos puntos como letras tenga
✓ ¡Puedes ROBAR la palabra desde el área de robo! (2 intentos por ronda)
✓ Si robas correctamente, ganas los puntos y termina la ronda

ÁREAS DE LA INTERFAZ:
📢 MENSAJES DEL JUEGO: Eventos, turnos, aciertos y errores
💬 CHAT NORMAL: Conversación entre amigos
🔫 ROBA PALABRA: Campo exclusivo para adivinar la palabra completa

SOLUCIÓN DE PROBLEMAS:
- Si no puedes conectar, verifica el firewall
- Asegúrate que todos estén en la misma red
- Prueba con la IP local (ipconfig en Windows, ifconfig en Linux/Mac)

¡DIVIÉRTANSE ROBANDO PALABRAS! 🎉
