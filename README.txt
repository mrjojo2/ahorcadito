🎮 AHORCADITO MULTIJUGADOR - GUÍA DE INSTALACIÓN 🎮

REQUISITOS:
- Node.js instalado (versión 14 o superior)
- Todos los dispositivos deben estar en la MISMA RED LOCAL

INSTALACIÓN:
1. Extraer todos los archivos en una carpeta compartida en red
2. Abrir terminal/CMD en esa carpeta
3. Ejecutar: npm install
4. Ejecutar: npm start

CÓMO JUGAR:
1. El servidor mostrará una IP (ej: http://192.168.1.100:3000)
2. Todos los jugadores deben acceder a esa IP desde su navegador
3. Cada jugador ingresa su nombre y se une al juego
4. Máximo 10 jugadores por partida
5. Por turnos, cada jugador intenta adivinar la palabra letra por letra
6. El que acierta suma 1 punto y se rota el turno
7. Si falla 7 veces, la ronda termina y otro jugador comienza

CARACTERÍSTICAS:
✓ Juego en tiempo real con hasta 10 jugadores
✓ Sistema de turnos rotativo
✓ Chat integrado para comunicación
✓ Puntuación persistente durante la sesión
✓ Más de 60 palabras diferentes

SOLUCIÓN DE PROBLEMAS:
- Si no puedes conectar, verifica el firewall
- Asegúrate que todos estén en la misma red
- Prueba con la IP local (ipconfig en Windows, ifconfig en Linux/Mac)

¡DIVIÉRTANSE!
